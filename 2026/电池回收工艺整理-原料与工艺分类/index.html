电池回收工艺整理——按原料与工艺分类

来源：乐享知识库电池回收工艺部文档（邦普 BRUNP 内部资料、学术论文、行业报告等）
整理日期：2026-06-14

一、按回收原料分类

1. 正极材料回收

原料类型 代表性物料 主要回收产物 主加工艺路线
三元正极粉 (NCM111/523/622/811, NCA) 镍钴锰酸锂、镍钴铝酸锂 NiSO₄、CoSO₄、MnSO₄、Li₂CO₃/LiOH → 三元前驱体 → 三元正极材料 湿法冶金浸出→萃取分离→合成
磷酸铁锂正极 (LFP) LiFePO₄、FePO₄ 再生 LFP 正极材料、Li₂CO₃ 直接修复再生（固态烧结/水热/共晶盐）
钴酸锂正极 (LCO) LiCoO₂、Co₃O₄ 再生 LCO、CoSO₄ 固态烧结/水热再锂化 + 退火
锰酸锂正极 (LMO) LiMn₂O₄ 再生 LMO 水热再锂化（0.1M LiOH 最佳）
纯钴电池废料 纯钴极片、纯钴粉料 CoSO₄ 酸浸→除杂→萃取

2. 负极材料回收

原料类型 代表性物料 主要回收产物 主加工艺路线
石墨负极 石墨 + 铜箔 再生石墨、Cu 箔 水洗除 SEI + 酸浸除金属杂质 + 高温退火
石墨增值转化 废石墨 + 铜箔 C/CuO 复合负极 原位氧化法（同步转化石墨和铜集流体）
石墨→Si-rGO 废石墨 多孔硅 + 石墨烯 → Si-rGO 复合负极 分离→制备多孔硅和 GO→静电自组装

3. 电解液回收

原料类型 代表性组分 主要回收产物 主加工艺路线
有机溶剂 EC、DMC、DEC、EMC、PC（占 80%+） 纯化碳酸酯溶剂 精馏提纯
锂盐 LiPF₆（占 15%，约 55万元/吨） LiPF₆ 晶体、Li₂CO₃、LiF、CaF₂ 直接回收法/锂盐转化法
添加剂 VC、FEC、PS 等 目前较少回收 含量低，尚未规模开展

4. 其他物料回收

原料类型 主要产物 工艺简述
电池浆料（含塑料袋、手套等异物） 电池粉 + NMP 浓缩液 破碎捞渣→絮凝沉降→液固分离→热处理→精馏
极片边角料 极片粉（正极/石墨粉）、铝渣、铜 多级破碎→分选→洗涤
隔膜（PE/PP） 回收隔膜 去离子水清洗→干燥
PVDF 粘结剂 回收 PVDF NMP/THF 溶解→沉淀回收，或 Cyrene 绿色溶剂分离
铜箔、铝箔 金属铜、金属铝 破碎分选→涡电流/比重法/色选

二、按工艺方法分类

1. 预处理工艺

1.1 放电

  • 方式：物理短路或盐水浸泡，释放残余电量
  • 目的：消除安全隐患，防止拆解过程中起火/爆炸
  • 标准：磷酸铁锂 ≤ 2.5V、三元锂 ≤ 3.1V

1.2 热解（焚烧）

  • 适用：软包电池、有机物含量高的物料
  • 工艺：将有机物（隔膜、粘结剂、电解液）燃烧殆尽
  • 产物特征：有机含量 < 2%，铜铝含量 ~10%，主金属含量略有降低

1.3 多级破碎 + 分选

  • 工序：粗破碎→多级破碎→筛分→风选→涡电流→比重法/摇床→色选
  • 分离目标：金属铜、金属铝、外壳、隔膜纸、电池粉（黑粉）
  • 非标设备为主，需根据物料特性定制

1.4 PVDF 粘结剂去除（关键瓶颈）

方法 特点
热解（~450°C 预烧结） 简单但产生 HF 腐蚀正极，微观结构可能破坏
NMP 溶剂溶解 有效但有毒、成本高
绿色溶剂（EG/Cyrene） 几秒钟无损剥离，可回收溶剂和 PVDF
水溶性粘结剂替代 未来方向，简化分离

2. 火法冶金（Pyrometallurgy）

项目 说明
原理 高温熔炼破坏电池材料，回收过渡金属合金
代表企业 比利时 Umicore（ValEas 工艺）
优点 工艺简单，适合大规模处理，可处理混合电池
缺点 能耗高（~0.7 kWh/kg）、CO₂ 排放高、锂回收率低（进入炉渣）、有机溶剂作为燃料烧掉
产物 Co(OH)₂/CoCl₂、Ni 合金、Mn 渣

3. 湿法冶金（Hydrometallurgy）⭐ 最主流路线

3.1 三元正极粉湿法回收流程（邦普工艺）

步骤① 浸出：

  • 低酸浸出：保持较低酸浓度，减少酸耗
  • 高酸浸出：确保反应彻底，降低残渣中贵金属含量

步骤② 除杂：

杂质 除杂方法 化学反应
Cu 铁粉置换 → 海绵铜 CuSO₄ + Fe → Cu + FeSO₄
Fe 调 pH → 黄钠铁矾沉淀 Na₂Fe₆(SO₄)₄(OH)₁₂ ↓
Al 调 pH → Al(OH)₃ 沉淀 Al₂(SO₄)₃ + Na₂CO₃ → Al(OH)₃ ↓
Cu/Cd 硫化沉淀 CuSO₄ + Na₂S → CuS ↓

步骤③ P204 萃取除杂：

  • 萃取剂 P204 逆流萃取，控制 pH 选择性除去 Li、Ca、Mg、部分 Mn
  • 萃取后水相进入精萃工序，有机相用硫酸再生循环

步骤④ P507 萃取分离 Ni/Co/Mn：

  • 控制 pH 选择性萃取分离 Ni、Co、Mn
  • 分别得到高纯度 NiSO₄、CoSO₄、MnSO₄ 溶液

步骤⑤ 提锂：

  • 锂中间体：沉淀富集得到的粗品碳酸锂/磷酸锂（含 Ca、Mg、重金属杂质）
  • 含锂废水：可溶性锂盐存在于水溶液中（浓度低、杂质高）

步骤⑥ 前驱体合成 + 三元材料烧结：

回收率： Ni ≥ 99.0%、Co ≥ 99.0%、Mn ≥ 99.0%、Li ≥ 85.0%

3.2 磷酸铁锂湿法回收

  • 酸浸出 + 除杂 → FePO₄ + Li₂CO₃
  • 经济性受锂价影响显著（2022年锂价高峰时毛利率约 17.89%）

3.3 钴冶炼工艺

四大类工艺：① 高温熔炼富集后湿法提取 ② 硫酸化焙烧后浸出 ③ 还原焙烧氨浸 ④ 加压浸出

  • 加压浸出：流程短、浸出率高、无有害废气，发达国家广泛采用

4. 直接回收（Direct Recycling）⭐ 前沿路线

核心理念：不破坏电极材料的原始结构，通过补充锂离子使废电极材料直接再生，保留材料制备过程中的大部分能量价值。

维度 火法 湿法 直接回收
能耗 ~0.7 kWh/kg ~4.5 kWh/kg ~0.2 kWh/kg
回收成本 ~$0.7/kg ~$0.4/kg ~$0.1/kg
产物价值 ~$1.5/kg ~$1.5/kg ~$8.2/kg
CO₂ 排放 高 中 低

4.1 固态烧结再锂化

正极类型 条件 性能恢复
LFP + Li₂CO₃, 650-700°C, N₂/Ar/H₂, 3-12h 147 mAh/g, 95% 容量保持率
LCO + Li₂CO₃, 800-900°C, air, 12h 152 mAh/g, 98% 循环保持率
NCM111 + Li₂CO₃, 800°C, 10h + 机械活化 165 mAh/g
NCM523 + Li₂CO₃, 850-920°C, O₂ 气氛 140-152 mAh/g

4.2 水热/溶液再锂化（⭐ 最通用）

  • 适用：LFP、LCO、NCM、LMO 均可
  • 介质：LiOH 溶液（4M 最常用，LMO 用 0.1M）
  • 条件：100-220°C, 2-20h → 后烧结 600-850°C
  • 特点：自饱和再锂化（多余的 Li⁺ 不会进入结构），更具通用性
  • LFP 案例：0.2M 柠檬酸 + 0.08M LiOH, 80°C + 600°C 烧结 → 159 mAh/g

4.3 电化学再锂化

  • 原理：以废正极为工作电极，在 Li₂SO₄ 电解液中通过外加电流驱动 Li⁺ 嵌入
  • 优点：低能耗、低成本
  • 缺点：需去除集流体和粘结剂，规模化有难度
  • LFP 案例：135 mAh/g, 500 次循环保持率 95.3%

4.4 共晶盐再锂化

  • 介质：LiOH/LiNO₃ 共晶（熔点 193°C）、LiCl-尿素（<120°C）
  • 优点：无质子环境（适合高镍 NCM）、自饱和再锂化
  • NCM523 案例：LiOH/LiNO₃ + CH₃COOLi, 400°C → 160 mAh/g

4.5 离子液体再锂化

  • 介质：[C₂mim][NTf₂] + LiBr, 150-250°C
  • 优点：无质子、不可燃、热稳定
  • 缺点：离子液体成本高，需回收循环使用

4.6 石墨直接回收

方法 工艺 效果
水洗法 LiCx + H₂O 反应去除残余 Li 和 SEI 去除杂质，简单环保
酸浸法 H₂SO₄ + H₂O₂ 去除 Ni/Co/Mn 杂质 纯度 99.9%
高温退火 N₂ 中 3000°C 石墨化 原子重排、结构恢复
B 掺杂升级 硼酸处理 + 退火 → B 掺杂石墨 接近理论容量，优于原始石墨

5. 电解液回收工艺

5.1 富集方法对比

方法 回收率 优点 缺点
冷冻法 低 安全环保、易控制 分离困难、能耗高
机械离心法 中 工艺简单、易工业化 能耗高、LiPF₆ 易分解、纯度低
溶剂浸取法 ~90% 能耗低、易控制分离 有机溶剂损耗、可能产生新污染
超临界 CO₂ 萃取法 >90% 回收率高、无毒廉价 设备要求高、投入成本大

5.2 锂盐回收

直接回收法：

  • 富集电解液 → 减压精馏分离溶剂 → 粗 LiPF₆ → 加 HF 重结晶提纯
  • 或：分析成分后补充溶剂/添加剂 → 重新配制成电解液

锂盐转化法（更成熟）：

目标产物 工艺 回收率
Li₂CO₃ 加 NaOH → 调节 pH → 加 Na₂CO₃ 沉淀 99.5%
LiF 加 KF/KHSO₄ 高温反应 -
CaF₂ 三级 Ca(OH)₂ 碱化 -

三、主要企业回收工艺

企业 工艺路线 特点
邦普 BRUNP（中国） 湿法冶金 + 定向循环 回收率 Ni/Co/Mn > 99%，Li > 85%，2019 年占全国回收量 30.9%
格林美 GEM（中国） 预处理→酸浸→分离提纯→再合成 超细钴粉和镍粉，电解液燃烧处理
Umicore（比利时） ValEas 火法工艺 高温熔炼回收 Co/Ni，有机溶剂作燃料
Recupyl（法国） 拆解→浸出→沉淀→净化 回收 Al、Co、Li，放弃回收电解液
OnTo（日本） Eco-Bat 超临界 CO₂ 工艺 专攻电解液回收
AEA（英国） 低温破碎→乙腈萃取→电沉积 回收 Cu、Al、CoO
ReCell Center（美国能源部） 直接回收研发 聚焦全组分直接回收，从制造到回收闭环

四、关键指标与挑战

经济性

  • 三元电池回收毛利率：约 23.95%（受钴镍价格下行大幅压缩）
  • 磷酸铁锂电池回收毛利率：约 17.89%
  • 2022 年锂价高峰后回收成本大幅上升，行业由”强卖方市场”转向
  • 直接回收中锂源成本占回收总成本的 95%

核心挑战

  1. PVDF 粘结剂去除：热解产生 HF 腐蚀正极，溶剂法有毒/成本高
  2. 正负极分离：浮选/重力分离效率不理想，石墨残留影响再生纯度
  3. 铝铜杂质残留：筛分无法完全去除，残留在产品中影响安全
  4. 电解液回收：挥发/易燃/有毒，LiPF₆ 遇水产 HF；规模小不经济
  5. NCM 系列混料：不同配方（111/523/622/811）混合后几乎无法分离
  6. 规模放大：实验室有效 vs 工业不稳定，产品一致性难保证

未来方向

  • 电池设计标准化 + 数字身份证（BIGP）→ 提高预分类效率
  • 水溶性粘结剂替代 PVDF → 简化分离
  • 机器人自动拆解 → 降低成本和安全风险
  • 升级回收（upcycling）：NCM111 → NCM622/811，废石墨 → Si-rGO
  • 固态电解质电池的回收体系预研
