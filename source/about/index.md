## 关于我

我是 **Gray2**，一名新能源制造业工程师。

选矿工程本科出身，现深耕锂电池材料体系、工艺参数优化与产线数据驱动的制造改进。兴趣横跨选矿、化工与新能源领域，相信工程思维的底层逻辑可以跨界迁移。

| 领域 | 具体方向 |

| 电池材料体系 | LFP / NCM / NCA / 钠离子 / 4680 大圆柱的性能、拆解与配方 |

| 跨界思考 | 从选矿到新能源，工程思维的迁移与演进 |

## 关于这个博客

这里是我记录技术思考和工作实践的地方。写博客的初衷很简单——**把散落在各处的知识系统化，把踩过的坑变成路标**。

文章分两个专栏：

- **电池技术材料体系**（11 篇）— 从正极材料全景图谱到下一代电池技术前沿，覆盖 LFP、NCM、钠离子、4680 大圆柱、电芯微观配方与选型实战
- **电池回收工艺**（3 篇）— 火法、湿法、直接回收三大路线详解，黑粉有价金属分布与毛利率深度分析
