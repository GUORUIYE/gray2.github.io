> 本文综合公开技术文献、行业报告及乐享知识库资料，系统梳理三元锂电池与磷酸铁锂电池在电芯层面的各组分重量占比，并延伸至子成分拆解与成本结构对比。

---

## 一、为什么关注"组分占比"？

锂电池的性能（能量密度、安全性、循环寿命）和成本，几乎完全由内部材料决定。理解各组分的质量占比，有助于：

- **回收端**：判断废旧电池中有价金属的品位，优化回收工艺路线
- **成本端**：定位成本敏感环节，指导采购策略
- **研发端**：识别材料替代的收益空间（如硅碳负极替代石墨、LMFP升级LFP）

## 二、电芯层面：两大体系重量占比总览

### 2.1 三元（NCM）锂电池

| 组分 | 重量占比 | 主要材料 | 说明 |
|:---:|:---:|:---|:---|
| **正极** | 30%–40% | 镍钴锰酸锂（LiNiₓCoᵧMn_zO₂） | 含铝箔集流体，是成本最高、价值最大的部分 |
| **负极** | 10%–15% | 石墨 / 硅碳复合材料 | 含铜箔集流体 |
| **电解液** | 10%–15% | LiPF₆ + 碳酸酯溶剂 + 添加剂 | 液态体系，重量波动主要来自注液量 |
| **隔膜** | 2%–4% | PP/PE 多孔膜 + 陶瓷涂层 | 重量轻但成本不低，高端产品涂覆Al₂O₃ |
| **集流体** | 8%–12% | 铝箔（正极侧）+ 铜箔（负极侧） | 铜箔较重，约占集流体总重的 60%–70% |
| **外壳及其他** | 10%–15% | 铝壳/钢壳/软包铝塑膜 + 极耳等 | 方形铝壳较重，软包较轻 |

> **典型参考值**（以方形 NCM523 电芯为例）：
> 正极 36% | 负极 14% | 电解液 12% | 隔膜 3% | 集流体 10% | 外壳等 25%

### 2.2 磷酸铁锂（LFP）电池

| 组分 | 重量占比 | 主要材料 | 说明 |
|:---:|:---:|:---|:---|
| **正极** | 30%–40% | 磷酸铁锂（LiFePO₄） | LFP密度低，正极片更厚，占比略高 |
| **负极** | 10%–15% | 石墨 | 与三元体系一致 |
| **电解液** | 10%–12% | LiPF₆ + 碳酸酯溶剂 + 添加剂 | 注液量通常略少于三元 |
| **隔膜** | 2%–3% | PP/PE 多孔膜 | LFP安全性好，部分可用干法隔膜 |
| **集流体** | 8%–12% | 铝箔（正极侧）+ 铜箔（负极侧） | 与三元一致 |
| **外壳及其他** | 15%–20% | 铝壳 + 极耳等 | LFP能量密度低，同容量电芯更重 |

> **典型参考值**（以方形 LFP 电芯为例）：
> 正极 37% | 负极 12% | 电解液 11% | 隔膜 3% | 集流体 10% | 外壳等 27%

## 三、正极材料内部拆解

正极是电池最核心也最贵的部分，内部还可进一步拆分。

### 3.1 三元正极（NCM）

| 子成分 | 占正极片重量比 | 说明 |
|:---:|:---:|:---|
| NCM 活性物质 | 90%–95% | 不同型号（523/622/811）镍钴锰比例不同 |
| 导电剂 | 2%–4% | 炭黑（Super P）、碳纳米管（CNT）等 |
| 粘结剂 | 2%–4% | PVDF（聚偏氟乙烯） |
| 铝箔集流体 | 另计 | 约 12–16 μm 厚 |

**NCM 正极活性物质的金属元素组成**：

| NCM 型号 | Ni : Co : Mn 摩尔比 | 特点 |
|:---:|:---:|:---|
| NCM111 | 1 : 1 : 1 | 早期产品，钴含量高，热稳定性好 |
| NCM523 | 5 : 2 : 3 | 当前主流，兼顾性能与安全 |
| NCM622 | 6 : 2 : 2 | 高镍过渡型 |
| NCM811 | 8 : 1 : 1 | 高端路线，能量密度最高，热管理难度大 |
| NCA | 8 : 1.5 : 0.5 (Ni:Co:Al) | 特斯拉曾主推，与NCM811同级别 |

> 以 NCM523 正极粉（不含集流体）为例，典型元素质量占比：
> - **Ni** ≈ 30%–33% | **Co** ≈ 12%–13% | **Mn** ≈ 9%–10% | **Li** ≈ 7%–8% | **O** ≈ 34%–38%

### 3.2 磷酸铁锂正极（LFP）

| 子成分 | 占正极片重量比 | 说明 |
|:---:|:---:|:---|
| LiFePO₄ 活性物质 | 88%–93% | 橄榄石结构，本征电导率低 |
| 导电剂 | 4%–8% | **明显高于三元**，需大量炭黑/CNT 补导电 |
| 粘结剂 | 2%–5% | PVDF 或水性粘结剂 |
| 铝箔集流体 | 另计 | 与三元一致 |

> LFP 正极粉元素质量占比：
> - **Fe** ≈ 34%–36% | **P** ≈ 19%–20% | **Li** ≈ 4%–5% | **O** ≈ 39%–41%

## 四、负极材料内部拆解

三元和 LFP 电池的负极体系基本一致，均以石墨为主。

| 子成分 | 占负极片重量比 | 说明 |
|:---:|:---:|:---|
| 石墨（天然/人造） | 90%–95% | 理论容量 372 mAh/g |
| 导电剂 | 1%–2% | 炭黑 |
| 粘结剂 | 2%–5% | CMC + SBR（水性体系） |
| 铜箔集流体 | 另计 | 约 6–8 μm 厚，占电芯重量 5%–8% |

> **趋势**：高端三元电池正逐步引入硅碳负极（Si/C 复合，硅含量 5%–10%），以提升能量密度，但循环膨胀问题仍需解决。

---

## 五、电解液内部拆解

| 子成分 | 占电解液重量比 | 说明 |
|:---:|:---:|:---|
| 锂盐（LiPF₆） | 12%–15% | 浓度通常 1.0–1.2 mol/L |
| 有机溶剂 | 80%–85% | EC + DMC + EMC + DEC 复配 |
| 添加剂 | 3%–5% | VC（成膜剂）、FEC、DTD 等 |

> 三元电池电解液注液量通常略多于 LFP（因高镍体系对电解液浸润性要求更高），但差异在 5%–10% 以内。

## 六、隔膜

| 项目 | 三元电池 | LFP 电池 |
|:---:|:---:|:---:|
| 基材 | PP/PE（湿法为主） | PP/PE（干法或湿法） |
| 厚度 | 16–20 μm | 16–25 μm |
| 涂覆 | 陶瓷涂覆（Al₂O₃）标配 | 部分不涂覆或水性涂覆 |
| 重量占比 | ~2%–4% | ~2%–3% |

---

## 七、两大体系对比一览

| 对比维度 | 三元（NCM） | 磷酸铁锂（LFP） |
|:---:|:---|:---|
| **正极占比** | 30%–40% | 30%–40%（LFP密度低，实际更厚） |
| **正极活性物质** | NCM（Ni/Co/Mn 三元协同） | LiFePO₄（橄榄石结构） |
| **导电剂需求** | 2%–4%（三元本征电导较高） | 4%–8%（**LFP本征电导率极低**） |
| **正极金属价值** | 高（Ni+Co+Li，钴价波动大） | 低（Fe+P，无贵金属） |
| **集流体** | 铝箔+铜箔 | 铝箔+铜箔（一致） |
| **电解液** | 10%–15%，注液量略高 | 10%–12%，略少 |
| **隔膜** | 湿法+陶瓷涂覆（高安全要求） | 干法/湿法均可 |
| **电芯重量能量密度** | 200–300 Wh/kg | 140–200 Wh/kg |
| **正极成本占比** | 35%–45% | 25%–30% |
| **电芯总成本** | 较高（受镍钴价格影响） | 较低（无贵金属依赖） |
| **安全性** | 热失控温度较低（~210°C） | 热稳定性极高（>500°C） |
| **循环寿命** | 1000–2000 次 | 3000–8000+ 次 |

## 八、回收视角的有价金属分布

| 电池类型 | 主要有价金属 | 典型含量（占电芯总重） | 回收价值 |
|:---:|:---|:---:|:---|
| **三元（NCM523）** | Ni | 7%–9% | 高 |
| | Co | 3%–4% | 极高 |
| | Mn | 2%–3% | 中 |
| | Li | 2%–3% | 高 |
| | Cu（集流体） | 5%–8% | 高 |
| | Al（集流体+外壳） | 3%–6% | 中 |
| **磷酸铁锂（LFP）** | Fe | 10%–14% | 低 |
| | Li | 1.5%–2% | 高 |
| | P | 5%–7% | 低 |
| | Cu（集流体） | 5%–8% | 高 |
| | Al（集流体+外壳） | 4%–7% | 中 |

> **关键差异**：三元电池回收的核心价值在 **Ni + Co + Li**，一吨黑粉的经济回报远高于 LFP；LFP 回收的核心价值在 **Li**，其余元素经济价值有限。这也是近年行业探索 LFP 优先提锂工艺的核心驱动力。

## 九、影响因素与数据声明

上述占比数据为行业典型范围，实际数值受以下因素影响：

1. **电池形态**：方形 / 圆柱 / 软包的外壳重量差异显著
2. **能量密度设计**：高能量密度版本正极占比更高
3. **工艺代际**：如极片压实密度提升会改变正负极占比
4. **NCM 型号**：523/622/811 的正极元素组成差异大
5. **应用场景**：储能型 LFP 更重视循环寿命，功率型更重视倍率，组分配比不同

> **数据来源**：综合自公开技术文献、行业研报（我的钢铁网/雪球/百度学术等）及乐享知识库内相关文档，部分数据为行业典型范围而非特定产品精确值，仅供参考。
