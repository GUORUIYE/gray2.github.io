## 引言

2025 年全球退役锂电池超过 100 万吨，2030 年有望突破 300 万吨。从废旧电池中回收有价金属，不仅是环保刚需，也是一门千亿级别的生意。目前产业上的回收工艺主要有三大流派——火法冶金、湿法冶金和直接回收。它们的技术逻辑、经济模型和环境影响截然不同。

本文基于邦普 BRUNP 内部工艺资料、ReCell Center 前沿研究及行业报告，系统拆解三大路线。

## 一、预处理：所有回收路线的起点

无论走哪条路线，废旧电池都需要经过预处理才能进入主工艺。预处理的核心工序：

### 1.1 放电
物理短路或盐水浸泡，将残余电压降至安全水平（LFP ≤2.5V，三元 ≤3.1V）。

### 1.2 多级破碎 + 分选
```
粗破碎 → 多级破碎 → 筛分 → 风选 → 涡电流分离 → 比重/摇床 → 色选
```
分离目标产品：金属铜、金属铝、外壳、隔膜纸、**黑粉（电池粉）**

预处理后得到的黑粉，就是后续主工艺的原料。

### 1.3 PVDF 去除（关键瓶颈）

粘结剂 PVDF 的去除是所有路线共同面临的首要挑战：

| 方法 | 特点 | 问题 |
|:---|:---|:---|
| 热解（~450°C） | 简单直接 | 产生 HF 腐蚀正极 |
| NMP 溶剂溶解 | 有效 | 有毒、成本高 |
| 绿色溶剂（EG/Cyrene） | 几秒无损剥离 | 尚在推广中 |
| 水溶性粘结剂替代 | 根本解决方案 | 需电池端配合改变配方 |

## 二、火法冶金：最古老也最简单

### 2.1 工艺原理

火法冶金的核心思路很简单：**直接高温熔炼，有机物当燃料烧掉，金属以合金形式回收。**

代表工艺——比利时 Umicore 的 ValEas 工艺：
- 将整块电池或黑粉投入高温炉（>1400°C）
- 有机物（电解液、隔膜、粘结剂）燃烧供能
- 过渡金属（Ni、Co、Cu）熔融为合金沉降
- Li、Mn、Al 进入炉渣

### 2.2 优缺点

| 维度 | 评价 |
|:---|:---|
| **优点** | 工艺简单粗暴、可处理混合电池、无需精细分选 |
| **能耗** | ~0.7 kWh/kg |
| **CO₂ 排放** | 高 |
| **锂回收率** | **低**（锂全部进入炉渣，需二次回收） |
| **产物价值** | ~$1.5/kg |
| **代表企业** | Umicore（比利时）、住友金属（日本） |

火法最致命的短板是锂回收率低——锂作为最有价值的元素之一大量进入炉渣。随着碳酸锂价格中枢上移，这一缺陷日益突出。

## 三、湿法冶金：当前最主流的路线 ⭐

### 3.1 邦普六步工艺

邦普 BRUNP（中国最大的电池回收企业之一，2019 年占全国 30.9% 回收量）的湿法冶金工艺是行业标杆。

**步骤① 浸出：**
```
正极粉 + H₂SO₄ → NiSO₄ + CoSO₄ + MnSO₄ + Li₂SO₄
（必要时加 H₂O₂ 做还原剂）
```
- 低酸浸出：减少酸耗，但反应不完全
- 高酸浸出：反应彻底，残渣中贵金属含量极低

**步骤② 除杂：**
| 杂质 | 方法 |
|:---|:---|
| Cu | 铁粉置换 → 海绵铜沉淀 |
| Fe | 调 pH → 黄钠铁矾沉淀 |
| Al | 调 pH → Al(OH)₃ 沉淀 |
| Cu/Cd | 硫化沉淀 → CuS ↓ |

**步骤③ P204 萃取除杂：**
利用 P204 萃取剂选择性除去 Li、Ca、Mg、部分 Mn

**步骤④ P507 萃取分离 Ni/Co/Mn：**
核心工序——通过控制 pH 选择性地将 Ni、Co、Mn 各自分离，得到高纯度 NiSO₄、CoSO₄、MnSO₄ 溶液

**步骤⑤ 提锂：**
沉淀富集锂 → 粗碳酸锂/磷酸锂 → 进一步精制

**步骤⑥ 前驱体合成 + 三元材料烧结：**
```
NiSO₄ + CoSO₄ + MnSO₄ → 三元前驱体 → + Li₂CO₃/LiOH → 三元正极材料
```

**回收率**：Ni ≥ 99.0%、Co ≥ 99.0%、Mn ≥ 99.0%、Li ≥ 85.0%

### 3.2 LFP 湿法回收

LFP 的湿法回收流程相对简单：
- 酸浸出 → 除杂 → FePO₄ + Li₂CO₃ 分离

由于 LFP 不含 Ni、Co 等高价值金属，回收经济性高度依赖锂价。当碳酸锂价格在 10 万元/吨以下时，LFP 回收基本不赚钱。

### 3.3 湿法工艺总评

| 维度 | 评价 |
|:---|:---|
| **优点** | 回收率高（Ni/Co/Mn >99%）、产品纯度高 |
| **能耗** | ~4.5 kWh/kg（高于火法） |
| **化学品消耗** | 大量酸碱，废水处理成本高 |
| **产物价值** | ~$1.5/kg |
| **代表企业** | 邦普、格林美、赣锋锂业 |

## 四、直接回收：最前沿的方向

### 4.1 核心理念

直接回收的思想是：**不破坏电极材料结构，通过补充锂使废电极直接再生**。保留材料制备过程中的大部分能量价值（合成能耗、结构依赖性性能等）。

火法/湿法本质上是"降级回收"——把材料打回金属盐/合金形态再合成；而直接回收是"同级回收"——材料在被消耗后直接回到可用状态。

### 4.2 三种工艺路线的全面对比

| 维度 | 火法 | 湿法 | 直接回收 |
|:---:|:---:|:---:|:---:|
| 能耗 | ~0.7 kWh/kg | ~4.5 kWh/kg | **~0.2 kWh/kg** |
| 回收成本 | ~$0.7/kg | ~$0.4/kg | **~$0.1/kg** |
| 产物价值 | ~$1.5/kg | ~$1.5/kg | **~$8.2/kg** |
| CO₂ 排放 | 高 | 中 | **低** |
| 回收率 | Co/Ni 高，Li 低 | 全元素 >85% | 取决于方法 |
| 产物形态 | 合金/炉渣 | 金属盐 | **再生材料** |

直接回收最大的优势在于**产物价值——再生正极材料可以直接回到电芯产线**，避免了火法/湿法"材料→金属盐→前驱体→烧结"的长链条价值损失。

### 4.3 六种直接回收方法

#### ① 固态烧结再锂化（最成熟）
向废正极粉中添加锂源（Li₂CO₃ 或 LiOH），在高温下烧结，恢复层状结构。
- LFP：+Li₂CO₃，650°C → 147 mAh/g，95% 容量保持率
- NCM523：+Li₂CO₃，850°C → 140–152 mAh/g
- 优点：操作简单、容易放大
- 缺点：需要高温、对混料体系适应性差

#### ② 水热再锂化（最通用）
在 LiOH 溶液（常用 4M）中 100–220°C 处理，Li⁺ 在水热条件下自发嵌入结构。
- LFP：0.2M 柠檬酸 + 0.08M LiOH，80°C + 600°C 烧结 → 159 mAh/g
- 覆盖：LFP、LCO、NCM、LMO 均可处理
- 优点：自饱和再锂化（多余的 Li⁺ 不会进入结构）

#### ③ 电化学再锂化
以废正极为工作电极，在 Li₂SO₄ 电解液中通过外加电流驱动 Li⁺ 嵌入。
- LFP：135 mAh/g，500 次循环保持率 95.3%
- 优点：低能耗、低成本
- 缺点：需去除集流体和粘结剂

#### ④ 共晶盐再锂化
利用低熔点共晶盐（如 LiOH/LiNO₃ 熔点 193°C）在较低温度下进行再锂化。
- 优点：无质子环境，适合高镍 NCM
- NCM523：LiOH/LiNO₃ + CH₃COOLi，400°C → 160 mAh/g

#### ⑤ 离子液体再锂化
以 [C₂mim][NTf₂] + LiBr 为介质，150–250°C 处理。
- 优点：无质子、不可燃、热稳定
- 缺点：离子液体成本高

#### ⑥ 石墨直接回收
- 水洗法去除 SEI → 简单环保
- 酸浸法去金属杂质 → 纯度 99.9%
- 高温退火石墨化 → 原子重排结构恢复
- **B 掺杂升级**：硼酸处理退火 → B 掺杂石墨，**性能超越原始石墨**

### 4.4 升级回收（Upcycling）

直接回收的一个更高阶形式——**将回收的低镍材料升级为高镍**：

- NCM111 → NCM622/811：通过补充 Ni 和 Li
- 废石墨 → Si-rGO 复合负极：原位氧化法转化

升级回收的经济价值更高，但技术难度也更大。

## 五、三大路线的战略定位

```
火法冶金：适合混合电池大规模处理，Co/Ni 回收为主，Li 损失高
        ↓
湿法冶金：当前主流，全元素高效分离回收，但化学品消耗大
        ↓
直接回收：前沿方向，保留材料全生命周期价值，正从实验室走向中试
```

**当前产业格局**：湿法为主（>70% 市场），火法为辅（成熟大厂），直接回收处于中试验证阶段。

**2030 年展望**：直接回收技术成熟后，将与湿法形成"分流回收"格局——高价值废料（NCM811、NCA）经直接回收再生后回到产线，低价值混杂料走湿法路线。
